# Presentations Archive

All presentations in this folder were archived on 2026-04-30 following the confirmed platform pivot to Solon Tax (ADR-018).

These presentations were created under the previous greenfield direction and advocate for Option C (greenfield build). They are no longer authoritative and must not be used to inform design decisions.

They are retained here as historical record only.

## Reason for Archiving

The platform pivot (ADR-018) supersedes the greenfield direction that all these presentations were built around. Specifically:

- All slide decks, talking points, and outline documents advocate for the greenfield stack (Flowable, Spring Boot monolith, standalone Keycloak, etc.)
- The `options-decision-talking-points.md` explicitly argues that Option C (greenfield) is the locked decision — this conclusion is superseded by ADR-018
- The `release-scope-table.md` and `requirements-coverage-summary.md` are framed against the greenfield architecture
- Architecture diagrams (`diag-*`) represent the greenfield design which is now under review

## What Supersedes These

See [`CLAUDE.md`](../../CLAUDE.md) and [`ADR-018`](../project-foundation/decisions/ADR-018-platform-pivot-solon-tax-confirmed.md) for the current direction.

A new design process is underway. New presentation material will be produced once the new architecture is settled.
